DailyGoods

A systemtic website to manage a large supershop easily.

About This Project

This project is a large system to run a small/large online supershop in one place. A strong secure database holding all the information of the stafs, products and stocks. The system provides services like product showing with description and online ordering facilities.
DailyGoods is an online shopping platform that offers a wide range of daily essentials, including groceries, beauty products, personal, and electronics care items. It features dynamic product display, secure user login, and responsive design ensuring a seamless and convenient shopping experience for users across all devices.

Technology Used

Languages used in this project- such as HTML, CSS, JavaScript, PHP and MySQL.
