Green EV Charging Station

A green world with pure energy.

About The Project

The Green EV Charging Station is a smart and eco-friendly electric vehicle (EV) charging solution. It utilizes solar power to charge EVs, switching to battery backup when solar power is insufficient. The system integrates Near Field Communication (NFC) technology for vehicle authentication and records the time and amount of current used during charging. This project offers a sustainable and efficient method for charging EVs, contributing to a greener environment.

The primary goal of the Green EV Charging Station is to provide an efficient, renewable energy-based charging option for EVs, reducing dependency on fossil fuels. Additionally, the system ensures secure and personalized charging through NFC technology and allows users to monitor charging data in real time.
